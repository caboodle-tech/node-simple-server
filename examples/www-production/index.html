Production Website/Application Demo

This is a simple demo of using Node Simple Server (NSS) for a static production website (HTML, CSS, JS). This is a good option for a web based internal work application. If NSS disconnects or crashes that will not be indicated to the user visually like in the website demo. You can still communicate with the backend via NSS websocket: NSS_WS.send(...)

You can also open the developer console to see helpful messages being printed by this demo.
